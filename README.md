# Google Translate App

A simple Python application that uses the `googletrans` module to translate text between languages.

## Setup

1. Create a virtual environment (recommended):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Install dependencies:

```powershell
pip install -r requirements.txt
```

## Run

```powershell
python translate_app.py
```

## Notes

- The app uses `tkinter` for the UI, which is included with most Python installations.
- If translation fails, check your internet connection and try again.
